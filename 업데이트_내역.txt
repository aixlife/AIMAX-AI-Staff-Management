AIMAX v1.0.11 / 2026-05-23
- 로컬 보안 설정 저장 시 기존 AI/API 키가 빈 값으로 덮이거나 삭제될 수 있는 경로를 차단했습니다.
- 이전 저장 과정에서 AI/API 키가 삭제 표시된 경우 다시 복구를 시도할 수 있도록 보정했습니다.
- 실행기가 이미 켜져 있을 때 웹에서 로컬 설정 열기를 다시 요청하면 기존 실행기에 정확히 전달되도록 보강했습니다.
- 로컬 설정 열기 지연/실패가 반복되면 자동 오류 보고에 필요한 진단 정보가 남도록 보강했습니다.

AIMAX v1.0.10 / 2026-05-23
- 같은 계정에서 Mac과 Windows 실행기를 함께 쓸 때 실행기 버전/업데이트 안내가 섞여 보일 수 있는 문제를 방지했습니다.
- 웹 대시보드에서 여는 로컬 보안 설정창을 네이버 ID/비밀번호 중심으로 정리했습니다.
- AI/API 키는 웹 설정 탭의 AI/API 연결에서 관리하도록 안내를 명확히 하고, 기존 로컬 AI/API 키는 삭제하지 않고 보존합니다.
- 비밀번호 입력 시 한/영 입력 상태를 다시 확인하도록 안내를 보강했습니다.

- 로컬 실행기 중복 실행 방지
- 예리 분리 앱 표시명을 블로그 글쓰기 기준으로 정리
