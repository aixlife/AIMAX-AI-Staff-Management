AIMAX v1.0.11 처음 읽어주세요

1. 실행 방법
- macOS에서는 AIMAX 앱(.app)을 실행합니다.
- 처음 실행하면 이용약관, 면책 조항, 개인정보 처리방침 동의 화면이 먼저 표시됩니다.
- 3개 항목에 모두 동의해야 프로그램을 시작할 수 있습니다.

2. 첫 실행 동의와 로컬 라이선스
- AIMAX는 1PC 1라이선스 / 영구 사용권 기준으로 제공됩니다.
- 첫 실행 동의가 완료되면 서버 인증이나 별도 키 입력 없이 로컬 라이선스가 자동 생성됩니다.
- 로컬 라이선스는 정품 사용 인식과 동의 기록을 위한 장치이며, 강한 복제 방지 장치가 아닙니다.
- 약관 버전이 변경되면 재동의를 요청할 수 있습니다.

3. 네이버 ID/PW와 AI/API Key 설정
- 웹 대시보드의 [로컬 설정 열기]는 네이버 ID와 비밀번호만 이 PC에 저장하는 용도입니다.
- Gemini, Claude, OpenAI, Apify 키는 웹 설정 탭의 [AI/API 연결]에서 관리합니다.
- 기존 로컬 실행기에 저장돼 있던 AI/API 키는 삭제하지 않고 유지됩니다.
- 블로그팀 로컬 작업에서 기존 로컬 AI 키가 필요한 경우 앱 안의 [직원 설정]에서 확인할 수 있습니다.

4. 기능 설명
- 고객 찾기: 키워드 또는 특정 블로거 링크를 기반으로 잠재 고객 블로거를 찾고 서로이웃 신청을 진행합니다.
- 소통: 이웃 새글을 기반으로 공감과 댓글 활동을 진행합니다.
- 글쓰기: 키워드 또는 Markdown 파일을 기반으로 블로그 글을 작성하고 발행, 예약 발행, 임시저장을 진행합니다.

5. 분리버전 앱 구분
- AIMAX-현주씨-영업사원: 고객 찾기 / 서로이웃 신청 중심 앱입니다.
- AIMAX-예리씨-블로그글쓰기: 공감 / 댓글 / 글쓰기 중심 앱입니다.

6. 외부 플랫폼 정책 주의
- 네이버 등 외부 플랫폼의 약관과 운영정책은 사용자가 직접 확인하고 준수해야 합니다.
- 과도한 서로이웃 신청, 댓글, 공감, 글 발행, 반복 문구 사용은 계정 제재, 검색 노출 제한, 활동 제한의 원인이 될 수 있습니다.
- AIMAX는 보수적 권장값을 제공하지만 계정 안전이나 성과를 보장하지 않습니다.
- 권장 한도를 초과하면 확인 절차가 표시되며, 초과 사용 결과의 책임은 사용자에게 있습니다.

7. 동시 실행 비권장
- 통합버전, 현주씨 앱, 예리씨 앱을 같은 네이버 계정으로 동시에 실행하지 않는 것을 권장합니다.
- 동시 실행은 로그인 충돌, 브라우저 세션 충돌, 과도한 활동으로 인식될 가능성을 높일 수 있습니다.
