    Intel Mac은 현재 지원하지 않습니다

    현재 Mac 설치 파일은 Apple Silicon(M1/M2/M3/M4) 전용입니다. Intel Mac에서는 실행기가 정상 실행되지 않을 수 있어 다운로드를 중단했습니다.

    처음 실행 전 확인

    Mac 설치 파일은 Apple Silicon(M1/M2/M3/M4) 전용입니다. macOS와 Windows가 보안 확인이나 권한 허용을 요청할 수 있습니다. Windows에서 설치 파일을 눌러도 아무 변화가 없으면 V3 Lite, SmartScreen, 회사 보안 프로그램의 보안 기록을 확인한 뒤 공식 AIMAX 설치 파일만 허용해주세요.

    설정 변경 안내

    AIMAX 설정 방식이 더 쉬워졌습니다

    1 예리 글 생성과 송이/윤미 같은 웹 기반 AI 작업은 웹 설정을 통해 더 안정적으로 사용할 수 있도록 개선되었습니다.
    2 Gemini, Claude, OpenAI, Apify 같은 AI/API 키는
    설정 > AI/API 연결에서 저장해주세요.
    3 기존에 실행기 로컬 설정에 API 키를 입력해두신 경우에는 기존 실행기 키 가져오기로 한 번만 옮기거나, 웹 보안 저장소에 직접 저장하면 됩니다.
    4 네이버 ID와 비밀번호는 기존처럼 내 PC의 로컬 실행기에만 저장됩니다.
    웹 서버로 보내지 않습니다.
    5 예리 서버 글 생성, Apify 수집, Gemini 분석처럼 외부 API 비용이 생길 수 있는 작업은 실행 전 확인창을 보여드린 뒤에만 진행됩니다.
